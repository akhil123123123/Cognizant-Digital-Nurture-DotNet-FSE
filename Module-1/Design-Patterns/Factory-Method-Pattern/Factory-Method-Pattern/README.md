# Factory Method Design Pattern

## Objective
To implement Factory Method Design Pattern in C#.

## Description
Factory Method Pattern is a creational design pattern.

It provides an interface for creating objects but allows subclasses to decide which class object should be created.

## Implementation

- Created Document interface
- Created WordDocument and PdfDocument classes
- Created abstract DocumentFactory class
- Implemented WordFactory and PdfFactory classes
- Objects are created using factory methods

## Technologies Used

- C#
- .NET

## Output

Opening Word Document

Opening PDF Document

## Conclusion

Factory Method Pattern helps in creating objects without exposing object creation logic to the client.
