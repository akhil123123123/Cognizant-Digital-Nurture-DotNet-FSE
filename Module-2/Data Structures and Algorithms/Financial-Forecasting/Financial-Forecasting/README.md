# Financial Forecasting

## Objective
To implement a financial forecasting application using recursion.

## Description
This project predicts the future value of an investment based on growth rate and time period.

The application uses recursion to repeatedly calculate the future value.

## Implementation

- Created recursive function
- Passed initial amount, growth rate and years
- Calculated future value using recursive calls
- Displayed forecast result

## Technologies Used

- C#
- .NET

## Algorithm Used

Recursion

## Time Complexity

O(n)

where n represents the number of years.

## Output

Financial Forecasting using Recursion

Initial Amount: 10000

Growth Rate: 10%

Years: 5

Future Value: 16105.1


## Conclusion
Recursion helps solve forecasting problems by breaking calculations into smaller repeated steps.
