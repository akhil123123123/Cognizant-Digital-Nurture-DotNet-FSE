# Ecommerce Search

## Objective
To implement search functionality for an ecommerce platform using Linear Search and Binary Search algorithms.

## Description
This project demonstrates how products can be searched efficiently using different searching techniques.

Linear Search checks each product one by one.

Binary Search divides the sorted product list and searches efficiently.

## Implementation

- Created Product class with ProductId and ProductName
- Stored products using Array
- Implemented Linear Search
- Implemented Binary Search
- Compared searching techniques

## Technologies Used

- C#
- .NET

## Time Complexity

### Linear Search
Best Case: O(1)

Worst Case: O(n)

### Binary Search
Best Case: O(1)

Worst Case: O(log n)

## Output

Linear Search Result:
Product Found: Tablet

Binary Search Result:
Product Found: Watch


## Conclusion
Binary Search performs better than Linear Search for large sorted datasets.
